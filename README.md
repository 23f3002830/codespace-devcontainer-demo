# Codespace Devcontainer Demo

This repository contains a devcontainer configuration for GitHub Codespaces with Python, uv, and FastAPI pre-configured.

## Features

- Python development environment via devcontainer
- `astral-sh.uv` and `ms-python.python` VS Code extensions
- FastAPI installed via `uv pip install` in postCreateCommand

## Usage

1. Clone this repository
2. Open in GitHub Codespaces or locally with Dev Containers
3. The environment will be automatically configured
